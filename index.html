問題5(配列)

y年m月d日が何曜日かはツェラーの公式を使って求められる。以下にツェラーの公式を示す。

h=(y+⌊y4⌋−⌊y100⌋+⌊y400⌋+⌊13m+85⌋+d)mod  7h=\left(y+\left\lfloor \dfrac{y}{4} \right\rfloor -\left\lfloor \dfrac{y}{100} \right\rfloor +\left\lfloor \dfrac{y}{400} \right\rfloor +\left\lfloor \dfrac{13m+8}{5} \right\rfloor + d\right)\mod 7 h=(y+⌊4y​⌋−⌊100y​⌋+⌊400y​⌋+⌊513m+8​⌋+d)mod7

なお、1月、2月は前年の13月、14月として扱う。つまり、2026年の1月なら2025年の13月となる。
hhhは曜日で次のように表せる。

曜日 日 月 火 水 木 金 土
h 0 1 2 3 4 5 6

また、⌊x⌋\left\lfloor x \right\rfloor⌊x⌋は床関数と言って、xxx以下の最大の整数を返すような関数で、floor(x)と書ける。
ここで、⌊x⌋\left\lfloor x \right\rfloor⌊x⌋のxが常に正のときはxの整数部分を返すような関数と考えられるので、整数型に変換するという操作で代替して良い。
つまり、ツェラーの公式はC言語では以下のように書き表せる。

int y,m,d;
int h = (y + y/4 - y/100 + y/400 + (13*m+8)/5 + d)%7;

なお、以降の小問ではとくに指示がなければ年は2026年とする。

  1. 1月から12月までの日数を配列に格納して、任意の月の日数を出力できるようなプログラムを作ってください。
  2. 0から6までの数字を入力したらそれに対応する曜日を出力するようなプログラムを作ってください。
  3. 月を入力として、その月の1日の曜日を数値で出力するようなプログラムを作ってください。
  4. 任意の月のカレンダーを出力するようなプログラムを作ってください。
    ただし、一行目は曜日として、日曜日始まりとする。月の1日が日曜日でない場合は空白文字を入れ、日付や曜日はタブ文字で区切る。なお、改行は\nで、空白文字はスペース、タブ文字は\tである。

  1. 構造体作って、1つの配列に格納し直してください。
  2. 各時刻におけるx2＋y2＋z2\sqrt{x^2＋y^2＋z^2}x2＋y2＋z2​の最大値を求めてください。なお、x2x^2x2はpow(x, 2)と書ける。
